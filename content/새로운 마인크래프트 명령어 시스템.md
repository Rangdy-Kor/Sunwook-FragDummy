# 새로운 마인크래프트 명령어 시스템

---

```toc
```

## 개요

---

가상의 마인크래프트 명령어 시스템의 사용법을 정리한 문서이다.

## 참고 사항

---

### 좌표

---

X+: 동쪽
X-: 서쪽
Y+: 위쪽
Y-: 아래쪽
Z+: 남쪽
Z-: 북쪽

### 권한

---

0: 참여자 (Guest) → 명령어 미사용 싱글 맵 권한, 기본 명령어만 사용 가능
1: 사용자 (User) → 서버 접속 시 부여되는 초기 권한의 기본값, 일반 명령어 사용 가능
2: 운영자 (Operator) → 명령 블록 및 함수의 권한, 서버 접속 시 부여되는 초기 권한의 한계값, 관리자 블록 상호작용 가능, 대부분의 명령어 사용 가능
3: 관리자 (Administrator) → 서버 주인 및 명령어 사용 싱글 플레이어의 인게임 권한, 타 플레이어에게는 콘솔을 통한 직접 부여만 가능, 서버 관리 명령어 사용 가능
4: 소유자 (Owner) → 서버 콘솔의 권한, 인게임 플레이어는 획득 불가능, 서버 통제 명령어 사용 가능
